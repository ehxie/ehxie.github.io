基于github搭建的个人博客

发表于2021-10-31更新于2021-10-31字数统计641阅读时长4分阅读次数

使用github和hexo快速搭建静态个人博客（最新版）

参考

前期准备

工具

1.NodeJS

2.Git

3.Hexo

  • Hexo，基于Node.js的博客框架，可以方便地生成静态博客页面，并部署在github上。

    • 作者是台湾的@tommy351，引用作者的话：hexo: A fast, simple & powerful blog framework, powered by node.js

仓库搭建

1.注册github

2.绑定ssh

3.创建仓库

  • 仓库名必须是：<username>.github.io
    • 例如我的名字是ehxie则我的仓库名是ehxie.github.io
  • 仓库必须是公开的

生成本地博客

新建一个文件夹(例如Blog)，进入该文件夹后，右键选择”git bash here”

发布到互联网

  • 配置_config.yml
    • 将仓库地址改为你自己的地址
    • 注意：type: git 中间需要有一个空格!!!其他的同理
      • github更新后master变成了main

仓库地址

安装自动部署发布工具

在.git目录下打开git命令行界面

  • 即上面步骤新建的文件夹(Blog)

生成令牌

登录github，点击个人头像旁边的小三角，点击settings，点击Developor settings，然后按下图操作

  • No expiration表示令牌不会过期(看个人选择)
  • 图中的4可以全选
  • 最后点击generate token

复制令牌，填到密码中

上传成功

  • 重新刷新下仓库，可以看到以下的内容

访问

  • 点击setting，复制仓库名，到浏览器上

  • 打开网页，输入仓库名回车即可访问

说明

这里只是说明了搭建博客的流程

  • 具体怎么编写博客，怎么自定义这里还没有涉及
